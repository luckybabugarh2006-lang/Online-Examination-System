# 📝 Online Examination System

A simple **Java console-based examination system** that allows users to log in, manage their profile, and take an online exam.

## 🚀 Features

* User Login
* Update Profile
* Change Password
* Online Exam
* Score Calculation
* Logout System

## ▶️ How to Run

```bash
javac OnlineExamination.java
java OnlineExamination
```

## 🛠️ Technology

* Java
